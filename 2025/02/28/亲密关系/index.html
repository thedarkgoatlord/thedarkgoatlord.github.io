对于亲密关系的思考

从小学五年级开始到现在，我曲折的感情史像一场大型的迷路，每当我决定一条分叉时，就既定了我的折返。于是总是一次次地回到原点，不时有些茫然。好处在于我经历了，并从中获得了成长。

所以，作为一个经历过恋爱大部分环节（暗恋、暧昧、热恋、异地、分手）并且每个环节都经历过不止一次的人，我想讲讲我的一些感悟，以及我对任何还没有经历过这些事情的人的一些由衷的建议。按照我的写作习惯，我先把本篇文章的结构列出来，然后再详细展开。

  1. 暧昧
  2. 热恋期
  3. 异地
  4. 面对分手
  5. 其它

暧昧

亲密关系通常是从暧昧开始的。你遇到了这个可能合适的人，然后决定做些什么，而暧昧就是从这里开始的。我会把暧昧描述为“没有进入关系但是想要进入”的情况，在经历一段上升期之后，取决于双方的意愿，暧昧可能会发展成亲密关系。但暧昧本身也能带来很美好的感受：在暧昧中，期待、不确定、不安……你的感官是被放大的。

这里，我最重要的观察是：如果要进入关系，你需要停止渴望关系。我的意思是，至少不要表现出来。道理其实很简单：自然和无企图性的互动比故意和有企图性的互动更能产生好感。一些换位思考可能会有帮助：如果你是对方，你有什么理由选择你自己？如果你觉得你简直完美，没有任何缺点，对方应该无条件地爱上你，那我也没法给你什么建议了。如果你还没有到这种需要重开的境地，那么我会建议：把你的主要精力放进你擅长的事情和你需要完成的事业当中，这会成为一种信号，即：我有自己独特而完整的生活，我是有吸引力的。而不是：我对你仰慕到五体投地，因此愿意追着你跑。

这里的一个比喻是行星。假设对方是一颗很大的行星，而你的目标是与对方相撞。如果你只是一颗小行星，那么你的命运大概率就是在对方的轨道上不断环绕，而没有任何靠近的能力（除非对方想要你的钱）。只有当你增加自己的质量，你与对方的引力逐渐增加的时候，你才有可能与对方相撞。

那么，假设你开始增加自己的行星质量了，那么，发展到什么程度之后才可以表白呢？我的答案是：当你已经基本确定你会表白成功时。听起来有点像废话，我承认。但是如果你并不确定表白的结果，你应该做的应该是想办法加深你与对方的联系和互相吸引，而不是在升温进行到一半时贸然将你和对方都推到必须做一个选择的境地。至少，我这么做过，失败了。这种“会成功”的直觉在关系发展到那一步时其实会变得较为明显，所以如果你一直没有这种这种感觉，也没感觉你眼下的关系有任何升温，我建议你及时止损，对方可能对你没有那么高的兴趣。

热恋期

这里真是没什么好说的，享受就好。可能有几点需要注意的：

  1. 问题解决。当问题出现时，一件很重要的事情是：不要把对方当作问题的来源。积极协商和对方一起解决问题，而不是指责。好吧，虽然在热恋期的人也很难注意到什么问题。
  2. 钱。钱的问题最好直接说，不要犹豫。
  3. 坦诚永远大于隐瞒。当你能够选择坦诚时，不要选择隐瞒。
  4. 表达。语言已经足以产生足够多的误解了，不要让你的爱意也被误解。也就是说，不要吝啬你的表达。这在一开始可能会很难以启齿，但是不要害怕迈出第一步。我会说这是很重要的。

为了保持新鲜感，可以试着去不时尝试一些新事物，学一些新东西，让你自己不陈旧。如果能一直保持热恋的感觉，那当然是很好的。当然，有时，如果你遇到了正确的人，每天都会是热恋，哪怕仅仅往复日常。我可以告诉你：这是真实存在的。

异地

异地是分手的一大原因，我觉得这里的关键是人的改变。尤其是在高中这段时间，我的心智、思想的内容以及平日里做的事情时刻都发生着较为大的变化，而一段异地的关系并没有办法很好地捕捉这种变化。因此，你已经变了，但对方还不知道，这对于一段关系来说是较为麻烦的，我会说。

面对分手

我看到过一句话：“分手时至少有一个人会受伤。”在我看来，这是对的。我受过伤，也伤过别人。头一回经历的感受是非常很强烈的：你可以想象一个人拿着铁勺从你的心脏挖下来一块，接着又一块，就这么一直挖，一直到你的心脏彻底转化为一地的血和肉块。然后你不得不俯下身，试着把所有东西捡起来，拼回去。

不过好处是，这种感受每个人大概只会经历一次。这或许是一种脱敏，也可能是因为从血泊里拼凑起来的心脏，往往更加强健，便能够在暴风雨中搏动。

总之，对现在的我来说，分手在大部分时候是一件令人难过，但是总的来说可以接受的事情：我把它看作是一种“我们原来不合适”的发现，而不是“我错过了适合的人”。

其它

我们经常会由各种各样的原因拒绝进入亲密关系。有时是忙碌，有时是“没兴趣”，有时是学校不允许，家长不允许。但是你其实知道，你真的有那么忙吗？你真的没有作为一个人类应有的本能的对亲密关系的渴求吗？学校不允许，家长不允许，但这是你的人生，不是吗？

诚实地面对自己对身心有益。
